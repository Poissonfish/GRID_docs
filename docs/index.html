Chapter 1 Installation

1.1 Windows

1.1.1 Install Anaconda

  • It’s recommanded to install GRID under Anaconda envirnoemnt, download it here0

  • Open the downloaded installer.

1.1.2 Install GRID

  • Open Anaconda Prompt

  • In the prompt, run the following command to install Rasterio, which is required by GRID.
conda config --add channels conda-forge
conda install rasterio
  • Use pip to install GRID
pip install photo_grid
  • Run GRID in Anaconda Prompt:
python -m grid

1.2 OS X

1.2.1 Install GRID

  • Install Python 3 and pip before installing GRID

  • Use pip to install GRID
pip install photo_grid
  • Run GRID in Terminal:
python -m grid
